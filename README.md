## Installation

To clone the repository, run the following command:

```sh
git clone https://github.com/gokhankatar/chat-app

``` 
go to directory

```sh
cd .\chat-app\

``` 
and then install npm:

```sh
npm install -f

``` 
open your browser:

```sh
npm run dev
